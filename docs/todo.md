### 408 专属注意事项
- 复杂度控制：尽量用 O(n) 时间、O(1) 空间的解法。
- 手写练习：LeetCode AC 后，在纸上默写一遍，注意边界条件和注释。
- 暴力解保底：想不出最优解时，先写暴力解，408 评分允许暴力解拿大部分分。
